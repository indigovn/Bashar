# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Repository Purpose

This repository contains transcribed and documented Bashar channeling sessions. The primary workflow is converting audio/video transcripts into comprehensive, structured study documents.

## Repository Structure

```
/                   # Word documents (.docx) - source/working files
/MD/                # Markdown documents (.md) - formatted study guides
```

## Document Format Standards

### Naming Conventions
- English: `Bashar_Topic_Name_Complete.md`
- Vietnamese: `Bashar_Topic_Name_Vietnamese.md`

### Document Structure
All markdown documents follow this template:

1. **Title**: `# Bashar: [Topic] - Complete Teaching`
2. **Overview**: Brief summary of the transmission's content
3. **Table of Contents**: Linked sections
4. **Content Sections**: Each major teaching with:
   - Blockquotes (`>`) for direct Bashar quotes
   - Tables for comparisons and frameworks
   - Code blocks for diagrams/flowcharts (using ASCII art with arrows ↓ →)
   - Bullet points for key principles
5. **Q&A Section**: If the session includes questions and answers
6. **Key Principles Summary**: Condensed takeaways organized by topic
7. **Closing Wisdom**: Final notable quotes

### Formatting Conventions
- Use `---` horizontal rules between major sections
- Use tables for frameworks and comparisons
- Use ASCII flowcharts in code blocks for processes
- Preserve exact quotes in blockquotes
- Vietnamese translations maintain identical structure to English originals

## Common Tasks

### Creating a New Document from Transcript
1. User provides transcript (audio/video transcription)
2. Identify main topics and teachings
3. Create structured markdown following the template above
4. Extract direct quotes for blockquotes
5. Create visual diagrams for processes/frameworks
6. Summarize key principles at the end

### Creating Vietnamese Translation
1. Read the English document completely
2. Create new file with `_Vietnamese` suffix
3. Translate all content maintaining identical structure
4. Keep proper nouns (Bashar, Atlantis, Anunnaki, etc.) untranslated
5. Translate section headers and table headers
